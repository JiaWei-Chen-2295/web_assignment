# 清爽现代UI设计说明

## 🎨 设计理念

### 配色系统
- **主色调**: `#10B981` (薄荷绿) - 清新活力
- **背景**: `#F8FAFC` 到 `#FFFFFF` - 纯净分层
- **文字**: `#0F172A` 到 `#94A3B8` - 清晰层次

### 视觉特征
- ✅ **大圆角**: `10-14px` 现代感
- ✅ **温柔阴影**: 分层柔和，不会突兀
- ✅ **微动效**: 悬停上浮、点击反馈
- ✅ **轻边框**: `1px` 细线分割

---

## 🔍 关键优化

### 1. 顶部工具栏
- 高度: `60px` → 更舒适
- 阴影: `shadow-sm` → 轻微悬浮感
- Logo居中对齐，增强识别度

### 2. 三栏布局
- 左侧: `260px` 知识树
- 中间: `280px` 文档列表
- 右侧: `1fr` 编辑区

### 3. 列表交互
```css
/* 悬停效果 */
.list-item:hover {
  background: var(--bg-hover);
  transform: translateX(2px); /* 轻微右移 */
}

/* 激活状态 */
.list-item.active {
  background: var(--bg-active); /* 薄荷绿淡背景 */
  border: 1px solid rgba(16, 185, 129, 0.15);
  color: var(--primary);
  font-weight: 600;
}
```

### 4. 按钮系统
- **默认**: 透明 + 细边框
- **悬停**: 浅灰背景 + 上浮
- **主按钮**: 薄荷绿 + 深色投影
- **删除**: 红色警告色

### 5. 编辑器
- 标题: `20px` 加粗，呼吸感
- 分屏: 玻璃感设计
- 预览: Markdown优雅渲染
  - 标题带下划线
  - 引用块用淡绿底色
  - 代码块深色背景

---

## 🔆 暗色主题

### 配色对比
| 元素 | 亮色 | 暗色 |
|------|------|------|
| 主背景 | `#F8FAFC` | `#0A0E14` |
| 面板 | `#FFFFFF` | `#0F1520` |
| 文字主 | `#0F172A` | `#F8FAFC` |
| 边框 | `#E2E8F0` | `#1E293B` |

### 暗色特点
- 深蓝黑背景，护眼舒适
- 薄荷绿更明亮突出
- 边框精细，防止"糊成一片"
- 保持所有良好对比度

---

## ✨ 交互细节

### Tooltip 提示
```html
<button data-tooltip="快捷键提示">💾</button>
```
- 出现: 悬停时上弹
- 位置: 元素上方居中
- 样式: 深色背景，白色文字

### Toast 通知
```javascript
this.showToast('操作成功', 'success');
```
- 动画: 右侧滑入
- 背景: 半透明 + 高斯模糊
- 边框: 彩色左侧条

### 搜索高亮
```css
mark.search-hit {
  background: #D1FAE5; /* 薄荷绿浅色 */
  padding: 1px 4px;
  border-radius: 4px;
  font-weight: 700;
}
```

---

## 🏗️ 组件模块化

所有CSS文件拆分，每个 < 200行:
- `base.css` - 变量与重置
- `layout.css` - 页面框架
- `components.css` - 组件样式
- `theme.css` - 暗色主题覆盖

---

## 🌟 视觉亮点

1. **Logo设计**: 使用点状符号作为图标
2. **虚线按钮**: "新建文档"使用虚线边框
3. **状态指示**: 动态光点（出版/草稿）
4. **模态框**: 磨砂玻璃效果 + 圆角
5. **滚动条**: 极细，仅在悬停时显示

## 📱 响应式

- >1024px: 完整三栏
- 768-1024px: 两栏（隐藏中间）
- <768px: 单栏（仅显示面板）
