# 快速开始指南

## 🌟 新架构说明

现在采用 **双页面架构**：
- **`index.html`** - 知识空间主页（显示所有工作区，支持排序）
- **`editor.html`** - 文档编辑器（点击工作区后跳转进入）

---

## 🚀 如何使用

### 第一步：打开主页
```
在浏览器打开:
file:///C:/Users/16010/Desktop/web_assignment/index.html
```

### 第二步：创建知识空间
1. 首次打开会看到欢迎页面
2. 点击"➕ 新建空间"或"立即创建"
3. 输入名称（如："学习笔记"）
4. 跳转到主页显示新创建的知识空间

### 第三步：排序（可选）
- 点击"最新" → 按创建时间排序
- 点击"名称" → 按A-Z排序
- 点击"数量" → 按书本数量排序

### 第四步：打开工作区
- **方式1**: 点击知识空间卡片（不点按钮）
- **方式2**: 点击卡片上的"打开"按钮
- **结果**: 跳转到 `editor.html` 进入编辑模式

---

## 📝 编辑器功能

### 跳转后的工作流程：

#### 面板1：左侧（工作区）
- 显示所有工作区可切换
- 点击切换查看不同工作区的内容
- 点击"+"

#### 面板2：中间（书本/文档）
- 点击书本标题展开/收起文档列表
- 点击文档打开编辑
- 右上角搜索框支持全局搜索
- "+" 按钮新建书本或文档

#### 面板3：右侧（编辑器）
- 实时Markdown编辑
- 实时预览（点击"预览"按钮）
- 保存（Ctrl+S）
- 发布/草稿切换
- 添加标签
- 导出文档
- 删除文档

### 返回主页
- **方法1**: 点击顶部工具栏最右侧的"🏠"按钮
- **方法2**: 快捷键 `Ctrl/Cmd + H`
- **方法3**: 直接修改URL为 `index.html`

## 🎨 设计原则

### 极简主义
1. **颜色**: 仅使用灰度 + 薄荷绿主题色
2. **动效**: 完全没有，保持操作瞬时反馈
3. **层次**: 通过边框和间距建立层级（无阴影）

### 操作流程
```
访问主页 → 排序/筛选 → 打开工作区 → 编辑文档 → 返回主页
     ↑                                                         ↓
     └─────────────────────────────────────────────────┘
```

## 🔧 遇到问题？

### 问题1：页面显示空白或报错
**解决方案**：
1. 打开浏览器控制台（F12）
2. 查看错误信息
3. 清空数据重试：
   ```javascript
   localStorage.removeItem('yuque-lite-data-v1');
   location.reload();
   ```

### 问题2：无法打开编辑器
**原因**：
- 可能没有先创建知识空间
- 浏览器不支持 Web Components

**解决方案**：
- 先在主页创建知识空间
- 使用现代浏览器（Chrome, Edge, Firefox, Safari）

### 问题3：数据丢失
**恢复方案**：
1. 点击主页右上角 🗂️ 按钮
2. 查看是否有自动备份
3. 选择备份点击"恢复"
